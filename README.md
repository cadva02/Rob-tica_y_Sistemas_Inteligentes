📌 Descripción del Repositorio

Este repositorio documenta el desarrollo técnico y los resultados obtenidos en el bloque TE3003B: Integración de Robótica y Sistemas Inteligentes. En colaboración con Manchester Robotics, se abordan temas de control avanzado, sistemas dinámicos y navegación autónoma utilizando ROS.
