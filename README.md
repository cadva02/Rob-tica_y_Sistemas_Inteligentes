📌 Descripción del Repositorio

Este repositorio documenta el desarrollo técnico y los resultados obtenidos en el bloque TE3003B: Integración de Robótica y Sistemas Inteligentes. En colaboración con Manchester Robotics, se abordan temas de control avanzado, sistemas dinámicos y navegación autónoma utilizando ROS.

🛠️ Entorno de Desarrollo

Para la ejecución de los algoritmos y simulaciones, se utilizó un entorno virtualizado con las siguientes herramientas:

    Sistema Operativo: Ubuntu con ROS preinstalado.

    Virtualización: VMware.

    Software Base: * VMware Workstation/Player

📚 Fundamentos Teóricos y Recursos
Sistemas Dinámicos y Control

El control de los sistemas robóticos se fundamenta en el análisis de sistemas lineales y no lineales:

    Modelado: Análisis del modelo dinámico de motores DC en MATLAB.

    Control Lineal: Fundamentos basados en Linear Systems Primer.

    Control No Lineal: Aplicación de control adaptativo y dinámico basado en las lecturas del MIT y textos de Khalil.

Robot Operating System (ROS)

La integración de los sistemas se realiza mediante el ecosistema ROS, enfocándose en:

    URDF: Modelado y descripción de robots (Unified Robot Description Format).

    TF2: Gestión de transformaciones de marcos de referencia y coordenadas.

    Tutoriales de Referencia: ROS TF2 y URDF Tutorials.
